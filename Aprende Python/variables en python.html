VARIABLES

Las variables de Python son nombres que se refieren a valores almacenados en la memoria. Se pueden asignar usando el signo igual (=), como x = 10 . Las variables de Python no tienen un tipo fijo, lo que significa que pueden almacenar diferentes tipos de valores en diferentes momentos. Por ejemplo, x se puede asignar a un valor de cadena más adelante, como x = "Hola". Para acceder al valor de una variable, simplemente use su nombre en una expresión o declaración.


Declaración y asignación: En Python, no necesitas declarar el tipo de variable explícitamente, simplemente asignas un valor a un nombre de variable y Python se encarga del resto. Para declarar una variable en Python, simplemente escriba el nombre de la variable seguido de un signo igual y el valor que desea asignar. Por ejemplo, x = 10 declara una variable llamada x y le asigna el valor 10.

var1 = 5
var2 = "Hola"


Tipado dinámico: Python es un lenguaje de tipado dinámico, lo que significa que el tipo de una variable puede cambiar en tiempo de ejecución. Por ejemplo, puede asignar un valor entero a una variable y luego asignar un valor de cadena a la misma variable. Python se encarga de la conversión de tipos automáticamente.

var = 10
var = "Hola"


Nombres de variables: Los nombres de variables pueden contener letras, números y guiones bajos, pero no pueden empezar con un número. Además, son sensibles a mayúsculas y minúsculas.

var = 10
Var = 20 Esto es diferente a la variable anterior.


Tipos de datos comúnes: Algunas de las variables más comunes en Python incluyen enteros (int), cadenas (str), listas (list), tuplas (tuple), diccionarios (dict), y más.

entero = 10
cadena = "Hola, Mundo"
lista = [1, 2, 3, 4]
tupla = (1, 2, 3, 4)
diccionario = {"clave":"valor"}


Variable de Alcance: El alcance de una variable determina dónde puede ser usada. En Python, las variables pueden ser locales (dentro de una función) o globales (en todo el código). Si una variable se declara dentro de una función, solo puede ser usada dentro de esa función. Si se declara fuera de una función, puede ser usada en cualquier lugar del código.

variableGlobal = "¡Soy global!"

def miFuncion():
variableLocal = "¡Soy local!"

Si intentas acceder a la variable local fuera de la función, obtendrás un error. Si intentas acceder a la variable global dentro de la función, Python buscará la variable en el ámbito local primero. Si no la encuentra, buscará en el ámbito global.
