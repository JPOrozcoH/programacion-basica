TIPOS DE DÁTOS

En programación, el tipo de dato es un concepto importante. Las variables pueden almacenar datos de diferentes tipos, y diferentes tipos pueden hacer diferentes cosas. Python tiene los siguientes tipos de datos incorporados de forma predeterminada, en estas categorías:

Texto: str

Numéricos: int, float, complex

Secuenciales: list, tuple, range

Mapeadores: dict

Arreglos: set, frozenset

Booleanos: bool

Binarios: bytes, bytearray, memoryview

Sin tipo: NoneType

DATOS DE TEXTO


STR (CADENAS DE TEXTO)

Los datos "Str" en Python se usan para almacenar y manipular texto. Son muy flexibles y permiten realizar muchas operaciones útiles con cadenas de caracteres.

Ejemplo de variables de texto:

nombre = "Juan"

saludo = "¡Hola mundo!"


Puedes concatenar cadenas (es decir, unirlas) fácilmente:

mensaje = saludo + " Soy " + nombre

print(mensaje) Esto imprime: "¡Hola mundo! Soy Juan"


También puedes usar el slicing para obtener partes de la cadena:

print(nombre[ 1:4]) Esto imprime: "uan"


Y hay una infinidad de métodos útiles para manipular strings, como cambiar a mayúsculas, buscar subcadenas, reemplazar caracteres, etc.

Por ejemplo:

texto = "Hola mundo"

print(texto. upper()) Esto imprime: "HOLA MUNDO"

print(texto. replace("mundo", "Python")) Esto imprime: "Hola Python"


Otra forma de formatear cadenas es usando las cadenas "f", o f-strings (cadenas de formato). Fueron introducidas en Python 3.6 y permiten incrustar expresiones dentro de cadenas utilizando llaves "{}".

Por ejemplo:

nombre = "Juan"

edad = 25

mensaje = f"Hola, {nombre}. Tienes {edad} años."

print(mensaje)


En este ejemplo, nombre y edad son variables, y al usar la cadena f f"Hola, {nombre}. Tienes {edad} años.", se reemplazan las expresiones dentro de las llaves con los valores actuales de esas variables. Al imprimir mensaje, obtendrás: Hola, Juan. Tienes 25 años.

La flexibilidad de las cadenas te permite trabajar con texto de muchas maneras creativas.

DATOS NUMÉRICOS


INT (NÚMEROS ENTEROS)

Los datos "Int" en Python son números enteros, ya sean positivos, negativos o cero, y no tienen decimales. Sirven para para contar cosas, hacer cálculos, y casi cualquier operación numérica.

Ejemplo de datos enteras:

x = 10

y = -5

z = 0


Python automáticamente detecta que estos datos son de tipo "int" cuando las asignas. Puedes hacer operaciones aritméticas con ellas:

resultado = x + y - z = 5


FLOAT (NÚMEROS DECIMALES)

Los datos "Float" representan números decimales. Son ideales cuando necesitas precisión con números decimales, como en cálculos científicos, financieros, o cualquier situación donde los decimales son cruciales.

Ejemplo de datos de punto flotante:

a = 3.14

b = -2.718

c = 0.0


Con los números de punto flotante se pueden realizar las mismas operaciones que con los números enteros.


Una característica importante de los "Float" es que pueden representar números muy grandes y muy pequeños gracias a la notación científica:

grande = 1.2e10: Equivale a 1.2 * 10 ** 10

pequeño = 1.2e-10: Equivale a 1.2 * 10 ** -10


Al igual que con los dátos "Int", Python reconoce los datos de tipo "Float".


DATOS COMPLEJOS (COMPLEX)

Los datos complejos representan números complejos, que tienen una parte real y una parte imaginaria. Estos son útiles en aplicaciones científicas y matemáticas donde se requieren cálculos con números complejos.

Un número complejo se define en Python con la forma a + bj, donde a es la parte real y b es la parte imaginaria.

Ejemplo de datos complejos:

x = 3 + 4j

print(x.real) Imprime la parte real: 3.0

print(x.imag) Imprime la parte imaginaria: 4.0

Puedes realizar operaciones aritméticas con números complejos de manera similar a como lo harías con números enteros o flotantes:

a = 1 + 2j

b = 3 + 4j

suma = a + b Resultado: (4+6j)

resta = a - b Resultado: (-2-2j)

multiplicacion = a * b Resultado: (-5+10j)

division = a / b Resultado: (0.44+0.08j)

Los números complejos son especialmente útiles en el análisis de señales, la física, y en la resolución de ecuaciones polinómicas y diferenciales.

DATOS SECUENCIALES


LIST (LISTAS)

Las listas en Python son colecciones ordenadas de elementos que pueden ser de diferentes tipos, incluso mezclando enteros, cadenas, y más. Son muy versátiles y se utilizan para almacenar una secuencia de elementos.

Ejemplo de listas:

lista = [1, "hola", 3.14, True]


Puedes acceder a los elementos de una lista mediante su índice:

print(lista[1]) Imprime "hola"


Es importante decir que, al momento de llamar un elemento de una lista mediante su índice, los valores inician en 0. Es decir que: la posición 0 es el primer elemento de la lista, la posición 1 es el segundo elemento de la lista, etc.


Y las listas tienen una gran cantidad de métodos útiles. Algunos de los más comunes son:

Añadir un elemento al final de la lista:

lista.append("nuevo") Añade la palabra "nuevo" al final de la lista; la lista quedaría así: lista = [1, "hola", 3.14, True, "nuevo"]


Eliminar un elemento específico:

lista.remove(3.14) Elimina el número 3.14 de la lista; la lista quedaría así: lista = [1, "hola", True, "nuevo"]


Insertar un elemento en una posición específica de la lista:

lista.insert(2, "insertado") Inserta el texto "insertado" en la posición #2; la lista quedaría así: lista = [ 1, "hola", "insertado", True, "nuevo"]


TUPLE (TUPLAS)

Las tuplas en Python son como las listas, pero inmutables. Una vez que las creas, no puedes cambiar sus elementos. Esto las hace más seguras y rápidas para ciertas operaciones. La principal diferencia entre una lista y una tupla es que para crear una lista, usamos corchetes cuadrados "[ ]", para crear una tupla usamos parentesis "( )".

Ejemplo de una tupla:

tupla = (1, "hola", 3.14, True)


En una tupla accedes a los elementos de la misma forma que en una lista:

print(tupla[1]) Imprime "hola"


Pero no puedes modificarla, Python arrojará un "TypeError":

tupla[1] = "adiós" Resultado: TypeError: 'tuple' object does not support item assignment


Las tuplas son perfectas para almacenar datos que no deben cambiar a lo largo del tiempo, como coordenadas (x, y), fechas (año, mes, día), etc. Son útiles en situaciones donde quieres asegurar que los datos no se modifiquen accidentalmente.


RANGO (RANGE)

los datos tipo "range" representan una secuencia de números y son comúnmente usados en bucles para iterar (repetirse) una cantidad específica de veces. Es muy eficiente y no almacena cada número en memoria, sino que los genera a medida que se necesitan.

Para ejemplificar esto, usaremos un ciclo "for", para ver más acerca de estos ciclos, visitar: Ciclos en Python, en el apartado de "Ciclos For".

num = 0

for num in range(5):

num += 1

print(num)

Lo anterior va a imprimir los numeros del uno al cinco.

DATOS DE MAPEO


DICT (DICCIONARIOS)

Los datos "Dict", o diccionarios, en Python son colecciones desordenadas de pares "clave" - "valor" ("key" - "value"). Esto significa que cada elemento en el diccionario tiene una clave única asociada a un valor, lo cual sirve para buscar valores rápidamente sin necesidad de recorrer toda una lista.

Ejemplo de un diccionario:

diccionario = {

"nombre": "Juan",

"edad": 15,

"mayor_de_edad": False

}


Para acceder a los valores, simplemente usas las claves:

print(diccionario["nombre"]) Imprime: "Juan"


Puedes agregar, modificar y eliminar elementos de un diccionario:

diccionario["carro"] = "Toyota" Añade un nuevo par "key":"value".

diccionario["edad"] = 18 Modifica el valor de la clave "edad"; pasa de ser 15 a 18.

del diccionario["nombre"] Elimina el par "nombre":"Juan"


Los diccionarios son útiles para almacenar y manejar datos estructurados y se usan en una variedad de aplicaciones, desde bases de datos hasta el procesamiento de datos.

CONJUNTOS


SET

Los datos "Set" en Python son colecciones desordenadas de elementos únicos, lo que significa que no puedes tener elementos repetidos en un set. Esto los hace útiles para eliminar duplicados y realizar operaciones matemáticas como la unión y la intersección.

Para crear un set, necesitas usar llaves "{ }"

Ejemplo de un set:

set = {1, 2, 3, 4, 5, 5} Recuerda que un set elimina los elementos duplicados, por consiguiente, el número 5 que está duplicado, se elimina. El resultado sería: 1, 2, 3, 4, 5.


Al igual que con las listas y diccionarios, con los set se pueden utilizar diferentes métodos, cómo añadir o eliminar elementos:

set.add(6) Agrega el número 6 al set.

set.remove(3) Elimina el número 3 del set.


Puedes realizar operaciones con otros sets:

otro_set = {4, 5, 6, 7}

union = set | otro_set Combina todos los elementos de los dos sets, eliminando duplicados. Resultado: {1, 2, 3, 4, 5, 6, 7}

interseccion = set & otro_set Encuentra los elementos comunes entre dos sets. Resultado: {4, 5}


FROZENSET

Los frozenset en Python son similares a los set, pero a diferencia de estos, los frozenset son inmutables. Esto significa que, una vez creado, no puedes cambiar su contenido.

Ejemplo de un frozenset:

mi_frozenset = frozenset([1, 2, 3, 4])

print(mi_frozenset)

Esto imprime: frozenset({1, 2, 3, 4})


Como los sets, los frozenset eliminan duplicados automáticamente. Puedes hacer operaciones típicas de conjuntos con ellos, como la unión, intersección y diferencia:

fs1 = frozenset([1, 2, 3])

fs2 = frozenset([3, 4, 5])


Unión:

union = fs1 | fs2

Resultado: frozenset({1, 2, 3, 4, 5})


Intersección:

interseccion = fs1 & fs2

Resultado: frozenset({3})


Diferencia:

union = fs1 - fs2

Resultado: frozenset({1, 2})


Los frozenset son ideales cuando necesitas garantizar la inmutabilidad y la integridad de los datos en tus conjuntos.

DATOS BOOLEANOS


BOOL (BOOLEANOS)

Los datos "Bool" en Python representan valores booleanos y pueden ser sólo uno de dos valores: "True" o "False". Son especialmente útiles en condiciones y bucles.

Ejemplo de variables booleanas:

es_mayor = True

es_menor = False


Las variables booleanas se usan comúnmente en declaraciones condicionales:

if 5 > 2:

print(True)

else:

print(False)


También son el resultado de operaciones comparativas:

a = 10

b = 5

print(a > b) Imprime True

print(a == b) Imprime False


Los booleanos son fundamentales para controlar el flujo del programa.

DATOS BINARIOS


BYTES

Los datos bytes representan secuencias de bytes, que son esencialmente cadenas de caracteres inmutables compuestas por valores enteros en el rango de 0 a 255. Son útiles para manejar datos binarios como archivos, imágenes y cualquier tipo de datos que no sean directamente texto.

Ejemplo de una cadena de bytes:

data = b"Hola, mundo"

print(data) Imprime: b'Hola, mundo'


Puedes acceder a los elementos de la misma manera que con una cadena o una lista:

print(data[0]) Imprime 72, que es el valor ASCII de 'H'

print(data[1:5]) Imprime: b'ola,'.


Es importante recordar que las cadenas bytes son inmutables, lo que significa que no puedes cambiar sus elementos directamente. Si necesitas modificar una cadena de bytes, debes convertirla primero en una lista de bytes , que es mutable.


BYTEARRAY

Los datos bytearray son muy similares a los bytes, pero con una diferencia clave: son mutables. Esto significa que puedes modificar sus elementos después de haberlos creado, lo que los hace extremadamente útiles para la manipulación de datos binarios en aplicaciones donde necesitas cambiar los datos directamente.

Crear un bytearray a partir de una cadena de bytes:

data = bytearray(b"Hola, mundo")

print(data) Imprime bytearray(b'Hola, mundo').


Puedes modificar elementos individuales del bytearray:

data[0] = 74 Cambia 'H' (72) por 'J' (74)

print(data) Imprime bytearray(b'Jola, mundo')


También puedes usar métodos para modificar el contenido de un bytearray:

data.append(33) Añade el byte correspondiente al carácter '!'

print(data) Imprime bytearray(b'Jola, mundo!')


Eliminar elementos:

del data[1:5] Elimina los bytes correspondientes a "ola,".

print(data) Imprime bytearray(b'J mundo!').


El bytearray es muy útil en situaciones donde necesitas trabajar con grandes cantidades de datos binarios y requieres eficiencia y flexibilidad en la manipulación de esos datos. Se utiliza frecuentemente en operaciones de E/S de archivos, procesamiento de datos de red, y en aplicaciones multimedia como imágenes y audio.


MEMORYVIEW

Los datos memoryview en Python permiten acceder a los datos subyacentes de un objeto de bytes sin copiar los datos reales. Son útiles para trabajar con grandes cantidades de datos binarios porque puedes manipularlos sin necesidad de duplicar la memoria, lo que ahorra tiempo y espacio.

Ejemplo para un memoryview:

data = bytearray(b"Hola, mundo") Se crea un bytearray

mem_view = memoryview(data) Se crea un objeto memoryview

print(mem_view[1:5]) Imprime b'ola,'

Modificar el contenido a través del memoryview:

mem_view[0] = 74 Cambia 'H' (72) por 'J' (74)

print(data) Imprime bytearray(b'Jola, mundo')


El memoryview permite trabajar directamente con los datos en memoria sin la sobrecarga de crear copias. Puedes acceder a partes del objeto, modificarlo, y realizar operaciones de manera eficiente.

Los memoryview son especialmente útiles en aplicaciones de alto rendimiento donde la gestión eficiente de la memoria es crucial, como en el procesamiento de grandes volúmenes de datos binarios en redes, archivos, o gráficos.

DATOS SIN TIPO


NoneType

NoneType es el tipo de dato que se asocia con el valor especial None en Python. None se utiliza para representar la ausencia de un valor, o "nada". Es especialmente útil en funciones que no devuelven un valor significativo, o para inicializar variables a un estado vacío o nulo.

Ejemplo basico de None

var = None


Puedes verificar si una variable es None usando el operador is:

if var is None:

print("La variable no tiene valor asignado.")


NoneType es útil para distinguir entre valores que están explícitamente establecidos a un estado vacío y aquellos que realmente contienen datos. Es una herramienta fundamental para manejar estados y flujos de control en tus programas.
